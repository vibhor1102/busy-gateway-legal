# Privacy Policy

Busy Gateway (“we”) processes WhatsApp message metadata and delivery status to send invoice/reminder notifications.

## Data We Process
- Phone numbers
- Message content/templates
- Delivery status (sent/delivered/read/failed)
- Operational logs

## Purpose
- Send transactional WhatsApp reminders and invoices
- Troubleshoot delivery issues
- Maintain message history and retries

## Retention
- Message history retained as configured in app settings
- Logs retained for operational diagnostics

## Sharing
- We do not sell personal data.
- Data is shared only with required service providers (e.g., Meta WhatsApp Cloud API) to deliver messages.

## Security
- Access is restricted to authorized operators.
- Credentials are stored in local configuration and should be protected by system access controls.
